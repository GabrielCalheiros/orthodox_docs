# ☦ Orthodox Docs

A small personal collection of documents, notes, and guides related to **Orthodox Christianity**.

This project was created to organize and preserve useful materials in a simple, accessible format.

---

## 📚 Contents

- **Komboskini & Chotki vs. Rosary & Terço**  
  Explanation of the Orthodox prayer rope and its differences from the Catholic Rosary.

- **Hesychasm**  
  Overview of the mystical tradition of inner stillness and the Jesus Prayer.

- **How to Set Up a Traditional Orthodox Prayer Corner**  
  Practical guide for creating a prayer space at home.

- **Books and Other Resources**  
  Curated reading list and external materials for further study.
